# DemandScience Glossary

- `SDM`: Service Delivery Manager
- `Template A`: week-1 weekly report layout and agenda
- `Template B`: week-2 weekly report layout and agenda
- `Weekly service review`: the customer-facing weekly report generated from the last 7 days of LogicMonitor data
- `Normalized bundle`: the stable report-ready LogicMonitor output returned by the MCP collector
